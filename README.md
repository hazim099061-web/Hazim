# Hazim Calculater — نسخة محسّنة

آلة حاسبة بسيطة ومُحسّنة بواجهة جميلة ومفيدة، مكتوبة بـ Python وTkinter.

## الميزات

- واجهة مظلمة مُنظّمة وأزرار مُجمّلة.
- عمليات أساسية: + − × ÷، النسب المئوية، القوى (^)، الجذر (√).
- أزرار إضافية: مسح (`C`)، حذف خانة واحدة (`←`)، تبديل الإشارة (`±`).
- دعم دوال رياضية من مكتبة `math` مثل `sqrt`, `sin`, `cos`, `log`, إلخ.
- اختصارات لوحة المفاتيح: أرقام، `Enter` للحساب، `Backspace` للحذف، `Esc` للمسح.

## المتطلبات

- Python 3.6+ (يأتي مع `tkinter` عادةً على معظم الأنظمة).

## طريقة التشغيل

1. افتح الطرفية في مجلد الملف.
2. شغّل الأمر التالي:

```bash
python "c:\\Users\\hp\\OneDrive\\Desktop\\Hazim Calculater.py"
```

## اختصارات سريعة

- أزرار: `C` لمسح، `←` لحذف آخر حرف، `=` أو `Enter` للحساب.
- يمكن كتابة `^` للقوة، و`√` لإدخال `sqrt(` تلقائيًا.
- `%` يحول القيمة إلى نسبة (يُحوّل `x%` إلى `x/100`).

## ملاحظات

- التعبيرات تُقيّم باستخدام `eval` مع مجموعة محدودة من دوال `math` لتقليل المخاطر.
- إن رغبت بإضافة زر ذاكرة (M+/M-/MR) أو تصدير سجل، أخبرني وسأضيفها.
